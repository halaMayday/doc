#### 2.disable这个rbd镜像不支持的特性：

```
-- rbd鏡像的一些特性，OS kernel並不支持，所以映射失敗。我們查看下該鏡像支持了哪些特性。可以看到特性feature一欄，由於我OS的kernel只支持layering，其他都不支持，所以需要把部分不支持的特性disable掉。 
[root@node2 ~]# rbd feature disable volumes/volume-ccb92381-63b8-4402-ad8b-17c79fc531a7  exclusive-lock object-map fast-diff deep-flatten
[root@node2 ~]# rbd info volumes/volume-ccb92381-63b8-4402-ad8b-17c79fc531a7rbd image 'volume-ccb92381-63b8-4402-ad8b-17c79fc531a7':
        size 2048 MB in 512 objects
        order 22 (4096 kB objects)
        block_name_prefix: rbd_data.196eb9c8f36f8
        format: 2
        features: layering
        flags: 
```
